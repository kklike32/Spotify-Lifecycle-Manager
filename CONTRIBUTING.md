# Contributing

Thanks for helping improve Spotify Lifecycle Manager!

## Development Setup

```bash
uv venv
uv sync
```

## Tests

```bash
uv run pytest -q
```

## Linting

```bash
uv run ruff check .
uv run black .
```

## Pull Requests

- Keep PRs focused and small
- Include tests for behavioral changes
- Update docs if behavior or configuration changes
